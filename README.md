# DocInsight RAG

## 📊 Project Description

DocInsight RAG is a Retrieval-Augmented Generation (RAG) system designed to analyze and query documentation generated by Agentic Coding tools (such as Cursor, Claude Code, and Kiro).

The system aggregates Markdown files from multiple tools into a unified knowledge layer and enables intelligent querying over the collected data.

It supports both:

* Semantic search (RAG)
* Structured data extraction for precise queries

---

## 🎯 Key Features

* Semantic search over documentation using embeddings
* Vector database integration (Pinecone)
* Workflow-based architecture (Event-driven)
* Query routing between:

  * Semantic retrieval (RAG)
  * Structured data extraction
* Source attribution for answers
* Interactive UI using Gradio

---

## 🧠 System Architecture

The system follows a hybrid approach:

* Semantic Search → for open-ended questions
* Structured Data → for list-based or precise queries
* Router → decides which path to use
